# Memory Card Game

A fun and interactive memory card matching game built with vanilla HTML, CSS, and JavaScript.

## Features

- 20 cards with 10 unique emoji pairs (planes, animals, hats, and more)
- Beautiful 3D card flip animations
- 4x5 grid layout
- Try counter to track your performance
- Smooth animations when matching pairs
- Victory modal displaying your final score
- Responsive design for mobile and desktop
- Play again functionality

## How to Play

1. Click on any card to reveal its emoji
2. Click on another card to find its matching pair
3. If the cards match, they disappear from the board
4. If they don't match, they flip back over
5. Try to find all 10 pairs in as few tries as possible
6. Your try count increases each time you flip 2 cards

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (ES6+)

## Getting Started

Simply open `index.html` in your web browser to start playing!
